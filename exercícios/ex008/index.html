Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Agora vamos aprender a trocar a cor do texto marcado, utilizando CSS.

Para isso dentro da tag MARK usamos a tag STYLE="BACKGROUND-COLOR": e atribuimos a cor desejada. Porém essa maneira serve apenas para a marcação desejada.

Para que a cor desejada fique gravada em todos os paragráfos desejados, devemos criar o BACKGROUND-COLOR no HEAD do script.

Pronto, aqui já temos nossa marcação gravada.
